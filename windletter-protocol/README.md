# windletter-protocol

Protocol data models and canonicalization interface:
- JWE/JWS/Payload POJOs, recipient headers, protected headers, WindMode enum.
- Recipient public info descriptor.
- JCS interface and default placeholder implementation.
Relies on core/crypto; used by armor/api/testkit.
